## Scope

This repository is **not** the paper itself.

- The full theoretical framework is described in the Zenodo paper.
- This repository serves as a space for:
  - experimental implementations
  - exploratory tools
  - applied prototypes
  - ongoing research notes

The purpose is to investigate how Noetica Theory
behaves when translated into computational systems.
